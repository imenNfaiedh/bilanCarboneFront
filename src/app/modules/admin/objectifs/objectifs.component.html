Objectifs du questionnaire

Ce questionnaire permet de collecter les données nécessaires à votre Bilan Carbone. Il vous aide à mesurer vos émissions de GES, identifier vos sources d’émissions, vous comparer à votre secteur et définir les actions prioritaires pour les réduire.

À l’issue de cette démarche, vous pourrez :

  • Piloter vos émissions GES
  • Réaliser votre propre Bilan Carbone
  • Interpréter les résultats
  • Valoriser ces résultats comme outil stratégique
